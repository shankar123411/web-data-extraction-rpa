# Web Data Extraction and Management using RPA with UiPath

## Overview
Automated web scraping workflows using UiPath Studio to extract 
structured data from multiple websites with error handling and 
retry logic.

## Key Features
- Multi-site web scraping with dynamic selectors
- Handles multiple page structures automatically
- Extracted data saved to Excel and CSV formats
- Robust error handling and retry logic
- Fault-tolerant, uninterrupted data extraction

## Technologies Used
- UiPath Studio
- Browser Automation
- MS Excel / CSV
- Error Handling & Retry Logic

## Results
- Significantly reduced manual data collection time
- Zero human error in data extraction pipeline
- Reliable end-to-end automation
